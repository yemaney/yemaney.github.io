## Book Recommendations

I personally believe that reading high quality books is critical to any learning experience.

This page is dedicated to highlighting some great books I have come across and would recommend to others.


---
### Fluent Python: Clear, Concise, and Effective Programming by Luciano Ramalho
![Fluent Python](https://images-na.ssl-images-amazon.com/images/I/41R+fNX-akL._SX379_BO1,204,203,200_.jpg){ align=left , width="200"}
!!! abstract

    Luciano Ramalho showcases what python has to offer. Teaches through approachable writing, code examples, and frequent notes on best practices.
    
    Focused on the core of python , along with its unique language features. This book will give a beginner the insight needed become an intermediate and advanced python programmer.

    My personal favorite part is how well the `Python Data Model` is explained. A language feature that is (in my opinion) the most important part about understanding how python works.

    Recommended to know the basics of python syntax before reading.

---
### The Linux Command Line: A Complete Introduction
![Fluent Python](https://images-na.ssl-images-amazon.com/images/I/51tfNcOQ75L._SX375_BO1,204,203,200_.jpg){ align=left , width="200"}
!!! abstract

    A book focused at introducing the linux command line and bash scripting to beginners. This book provides a lot of value even if you are not going to be a system administrator.

    Learning how to, move through the command line, search using path expansions, use pipes and filters, edit and search files, and make simple bash scripts are skills that can make any developer more productive.

    My favorite part is learning how everything is a file in the unix system, which enables the functionalities such as pipes and filters.
    
    No previous knowledge required, only a terminal to practice with the book.
